CREATING THE SUPERBIOUS TOKEN ON THE GOERLI TESTNET

    • Prerequisites:

  • Basic understanding of blockchain technology and Ethereum.
  • Ethereum Remix IDE (for writing and deploying smart contracts).
  • MetaMask wallet (for interacting with the Ethereum blockchain).
  • Etherscan account (for monitoring and verifying contract deployment).

    • Step 1: Set Up Development Environment:

  • Install MetaMask browser extension and create an Ethereum wallet.
  • Access the Ethereum Remix IDE and connect it to your MetaMask wallet.

    • Step 2: Writing the Smart Contract:

  • Open the Remix IDE and create a new file named `SuperbiousToken.sol`.
  • Define the `ERC20` token standard by importing the necessary interface.
  • Create a new contract named `SuperbiousToken` that implements the `ERC20` interface.
  • Define the constructor to set the initial token parameters (name, symbol, decimals, total supply: 100 million tokens).
  • Implement functions for `balanceOf`, `transfer`, and `approve`.

    • Step 3: Deploying the Smart Contract:

  • Compile the smart contract in the Remix IDE.
  • Switch to the "Deploy & Run Transactions" tab and select the Goerli testnet environment.
  • Deploy the smart contract by clicking the "Deploy" button. Confirm the transaction in MetaMask.

    • Step 4: Interacting with the Token:

  • Access your deployed contract address on the Remix IDE or MetaMask.
  • Add your token to MetaMask by clicking "Add Token" and entering the contract address.
  • Use the Remix IDE or Web3.js to interact with your smart contract functions programmatically.
  • Transfer tokens between accounts using the `transfer` function.
  • Use Etherscan to monitor transactions, verify your contract, and explore token details.

    • Final Result:

    The "Superbious" token has been successfully created and deployed on the Goerli testnet of the Ethereum blockchain. With a total supply of 100 million tokens, it adheres to the ERC20 token standard and can be interacted with using the provided smart contract functions. Users can transfer "Superbious" tokens between accounts and explore its details through Etherscan. The address of the Superbious token on Etherscan is 0x2D73c9FCA19db8E644378E03AAa1E685Ab424D62.

    Conclusion:

    Creating the "Superbious" token on the Goerli testnet of the Ethereum blockchain demonstrates the process of deploying a cryptocurrency token using smart contracts. By following the step-by-step process outlined in this project report, you have gained hands-on experience in blockchain development and have successfully created and interacted with a custom token. This project provides a foundational understanding of token creation, smart contract deployment, and the Ethereum ecosystem.
